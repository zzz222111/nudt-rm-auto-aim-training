Detailed Description

ROS2 component implementation for fusing detected armors into a stable tracked target.

Responsibilities implemented here:

  • Subscribe to /detector/armors with TF filtering and transform detections into the target_frame.
  • Drive the Tracker lifecycle: init/update, publish debug measurements and final Target messages.
  • Provide RViz markers for visualization of the tracked target and per-armor positions computed from the fused state.
