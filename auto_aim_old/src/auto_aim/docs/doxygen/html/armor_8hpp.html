Detailed Description

Data structures for lights and armors used by the armor detector.

This header defines lightweight POD-style containers that represent detected light bars (Light) and paired armor plates (Armor). They store image geometry (centers, corners), evaluation metrics (length, width, tilt angle), and classification outputs (number image, label, confidence).

Purpose and usage:

  • Light holds the rotated rectangle returned by OpenCV and derives convenient points such as top/bottom, length and width.
  • Armor pairs two lights, stores the computed center and the result of the number classification stage. The type field indicates small/large.
