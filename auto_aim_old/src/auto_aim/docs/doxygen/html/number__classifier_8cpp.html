Detailed Description

Warp number ROIs and run an ONNX DNN to classify armor digits.

Responsibilities:

  • Load an ONNX model and text labels on construction.
  • extractNumbers: warp the image patch defined by the two lights into a canonical cropping for classification.
  • classify: run the network, compute softmax probabilities, fill the armor fields, and filter low-confidence / ignored classes.
