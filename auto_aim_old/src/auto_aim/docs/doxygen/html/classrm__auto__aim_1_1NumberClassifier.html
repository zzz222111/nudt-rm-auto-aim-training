Detailed Description

Wrapper around an ONNX DNN for recognizing the digit on an armor.

The classifier holds a loaded network and label list. Usage pattern:

  1. Call extractNumbers() to warp/crop number ROIs into canonical size.
  2. Call classify() to run the network and populate Armor::number and related fields.
