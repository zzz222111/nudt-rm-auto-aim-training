Detailed Description

Small Extended Kalman Filter helper used by the tracker.

This EKF implements a generic interface where the nonlinear process and measurement functions (f, h) and their Jacobians are provided as std::function objects. The filter stores prior/posterior covariances and exposes predict() and update() methods used by the Tracker.

This design keeps the filter reusable for different state/measurement dimensions while allowing the caller to supply process/measurement noise generators.
