Detailed Description

ROS2 node component that runs the armor detection pipeline.

Subscribes to camera images, runs detection/classification, computes PnP poses and publishes jlcv_interfaces::msg::Armors and visualization markers. When debug is enabled, publishes intermediate images and debug messages.
