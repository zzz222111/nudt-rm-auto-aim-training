Detailed Description

Implementation of the image processing pipeline for finding lights and matching them into armor candidates.

Key steps implemented here:

  • preprocessImage: extract target-colored bright regions and produce a binary mask.
  • findLights: extract contour-based rotated rects and filter them to plausible light bars.
  • matchLights: pair lights to form armor hypotheses and perform basic geometric checks.
  • Debug helpers (drawResults, getAllNumbersImage) to visualize intermediate outputs.
