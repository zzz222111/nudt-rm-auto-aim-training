Detailed Description

Main detection pipeline class.

Responsibilities:

  • Preprocess incoming RGB frames into a binary mask for the target color.
  • Find individual light-bar candidates and pair them into armor hypotheses.
  • Maintain debug outputs and forward number classification to the NumberClassifier.
