Detailed Description

ROS2 component node declaration for the armor detector.

This node subscribes to camera images, runs the Detector pipeline to locate armors, uses PnPSolver to estimate 3D poses, and publishes jlcv_interfaces::msg::Armors and RViz visualization markers. It also exposes debug topics (binary image, number images, debug messages) when the debug parameter is enabled.
