Detailed Description

Solve PnP for detected armors and simple image-center distance.

PnPSolver converts 2D image coordinates of armor vertices to a 3D pose in the camera frame using OpenCV's solvePnP. It stores canonical 3D model points for small and large armors and provides a helper to compute pixel distance to the camera principal point.
