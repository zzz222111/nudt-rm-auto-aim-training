Detailed Description

ROS2 component implementing the armor detection node.

This source implements the logic declared in detector_node.hpp. It handles parameter initialization, subscriptions to camera images and camera info, sets up the Detector and PnPSolver, and publishes detected armors as jlcv_interfaces::msg::Armors along with visualization markers and debug image topics.
