Detailed Description

Detection pipeline for extracting light bars and matching armors.

The Detector class encapsulates the image preprocessing, light candidate extraction, light-pair matching to armors, and debug instrumentation. It also holds a NumberClassifier instance for extracting/recognizing the digit on the armor plate.

Contract (inputs/outputs):

  • Input: RGB image (cv::Mat) via detect()
  • Output: vector<Armor> containing detected armor candidates with associated number image and classification data.
