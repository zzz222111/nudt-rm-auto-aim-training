# 第一次作业——用python、opencv实现小球视觉识别与跟随

## 作业简介

本作业借助AI辅助，使用 Python + OpenCV 实现视频内绿色球体检测、跟踪与运动可视化。通过 HSV 颜色分割结合轮廓几何特征筛选绿球，使用箭头标注运动方向与速度大小，同时展示速度、运动状态、帧号等信息；支持目标短时丢失轨迹预测，保证跟踪连续性。

## 主要功能

    目标检测：基于 HSV 颜色范围提取绿色区域，结合面积、圆度、长宽比过滤干扰物体，精准识别绿球。
    目标跟踪：计算相邻帧位移得到运动速度，结合位置预测完成目标匹配，提升跟踪稳定性。
    可视化标注：
        圆圈标记绿球位置，红点标记球心；
        箭头方向代表运动方向，箭头长度代表运动速度（速度越快，箭头越长）；
        文字展示：实时速度、加速 / 减速 / 匀速状态、当前帧序号；
        短时丢失：基于速度预测位置并标注 green ball predicted；完全丢失标注 green ball lost。

## 可调参数说明

    K_MAX_LOST_FRAMES：目标最大允许丢失帧数，超过该值则终止跟踪。
    HSV 绿色阈值：lower_green / upper_green，绿球颜色不符时可调整此参数。
    轮廓筛选阈值：面积、圆度、长宽比，适配球体靠近画面边缘、透视形变场景。
    箭头缩放范围：限制箭头长度在 25~140 像素，避免显示异常。

## 输出说明
程序运行结束后，在指定路径生成 MP4 格式标注视频，包含绿球框、运动箭头、速度状态、帧号等全部可视化内容。
